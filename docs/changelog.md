# Changelog

Логи версий ведутся начиная с `1.0.0`.

## [1.0.0] - 2026-07-14

### Added
- Асинхронная обертка над Argon2
- Методы `hash` и `verify`
- Поддержка кастомных параметров
- 100% покрытие тестами
- Полные docstrings
- Документация mkdocs

## [1.1.0] - 2026-07-14

### Added
- Возможность не писать префикс `argon2__` в параметрах
- Обратно совместимо и будет работать и с префиксами
